# 2438번 :  별 찍기 - 1

## 문제

첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

## 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 예제 입력 1

```
5

```

## 예제 출력 1

```
*
**
***
****
*****
```

---

처음 제출한 정답 

```sql
N = int(input())
1<= N <=100
    
for i in range(N):
    k = len(i)
    print(k*"*")
```

 그 다음

```sql
N = int(input())
1<= N <=100
    
for i in range(1,N+1):
    print("*"*i)
```
